# Boonyisa Shop - Flask E-commerce Website

เว็บไซต์ขายของออนไลน์ที่สมบูรณ์ด้วย Python Flask พร้อมฟีเจอร์ครบครัน

## ฟีเจอร์หลัก

- ✅ **หน้าแรก** - แสดงสินค้า 6 รายการพร้อมรูปภาพสวยงาม
- ✅ **ระบบตะกร้า** - เพิ่ม/ลบสินค้าและแสดงจำนวนในตะกร้า
- ✅ **ตัวกรอง** - กรองสินค้าตามหมวดหมู่ (คอมพิวเตอร์, กล้อง)
- ✅ **หน้าชำระเงิน** - ฟอร์มกรอกข้อมูลลูกค้าและวิธีการชำระเงิน
- ✅ **รูปภาพสินค้า** - สร้างอัตโนมัติ 6 รูปสินค้าตัวอย่าง
- ✅ **ออกแบบสวย** - UI/UX ดีงาม responsive บนมือถือและเดสก์ทอป

## สินค้า

## การติดตั้ง

1. ติดตั้ง dependencies:
   ```
   pip install -r requirements.txt
   ```

2. รันแอปพลิเคชัน:
   ```
   python app.py
   ```

3. เปิดเบราว์เซอร์ไปที่ `http://localhost:5000`

## โครงสร้างโปรเจกต์

```
boonyisa-shop/
├── app.py                 # แอปหลัก Flask
├── requirements.txt       # Dependencies
├── templates/
│   ├── index.html         # หน้าแรก
│   └── checkout.html      # หน้าชำระเงิน
└── static/
    ├── css/
    │   └── style.css      # สไตล์ CSS
    ├── js/
    │   └── script.js      # JavaScript
    └── images/            # รูปภาพสินค้า (สร้างอัตโนมัติ)
```

## เทคโนโลยีที่ใช้

- **Flask**: Web framework
- **Pillow**: สร้างรูปภาพอัตโนมัติ
- **Bootstrap 5**: Responsive design
- **HTML/CSS/JavaScript**: Frontend

## การทำงาน

- รูปภาพสินค้าจะถูกสร้างอัตโนมัติเมื่อรันแอปครั้งแรก
- ใช้ session เพื่อเก็บข้อมูลตะกร้า
- หน้า checkout มีฟอร์มสำหรับกรอกข้อมูลและเลือกวิธีชำระเงิน
- ออกแบบ responsive ทำงานได้ทั้งมือถือและเดสก์ทอป
